At Newmark, I worked on several web applications used by brokers and research teams to manage real estate data and deals. One of my main projects was a Commercial Real Estate stacking plan that visualized tenant and vacancy data floor by floor. I built it with React and TypeScript, adding filters, interactive layouts, and color-coded lease expirations to make portfolio insights easier to interpret.

Toward the end of my internship, I designed a responsive GIS landing page with Tailwind CSS that showcased site selection services through modular components and integrated form submissions. Working across these projects helped me strengthen my understanding of scalable component design, state management, and how small UI changes can meaningfully improve daily workflows.

At Everise, I worked on automating data workflows and improving how performance metrics were tracked. I built ETL pipelines using Python, SQL, and Apache Airflow to combine data from different client systems into a central warehouse. Once the data was integrated, I created Power BI dashboards with DAX to visualize key metrics like Bill-to-Pay percentage and service-level performance. At eMeasurematics, I worked on predictive maintenance software for industrial vehicles in steel facilities.
